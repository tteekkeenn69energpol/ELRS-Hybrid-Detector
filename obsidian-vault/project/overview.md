## Що це

Розробка пристрою на базі 6 технологій з використанням мультиагентної системи **Claude Code**.  
Агенти запускаються через термінал (CLI), паралельно, кожен зі своїми скілами та правилами.

## Поточний фокус

**Stage 1 — 2D OS-CFAR standalone модуль**

Стан реалізації на старті:
- 60% коду написано
- 10% тести
- 20% інтеграція
- → Ціль: довести до повністю робочого стану

## Архітектура агентів

```
Оркестратор
    │
    ├── Knowledge Builder  (одноразово)
    ├── DSP Research       (алгоритми, теорія)
    ├── C++ Dev            (реалізація)
    ├── Test / QA          (блокуючий!)
    └── Docs               (пам'ять системи)
```

## Покроковий цикл

```
Крок N:
  Оркестратор призначає
       ↓
  Агент виконує в терміналі
       ↓
  Артефакт → Obsidian + GitHub
       ↓
  Test/QA підтверджує
       ↓
  Docs синхронізує
       ↓
  Крок N+1
```
