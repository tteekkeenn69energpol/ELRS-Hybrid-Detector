## Фінальна архітектура (approved-v8)

```
IQ buffer (50ms) + t_offset (з PMT Stage 1)
    │
    ├─→ cwt_bw_estimation(iq, t_offset) ─────────────────────
    │     Welch PSD: iq[t_offset:t_offset+65536]              │
    │     N_fft=4096, K=31 Hann frames                        │
    │     per-class score[b]=mean(S[0..b/2])/mean(S[b/2..b]) │
    │     → bw_candidate ─────────────────────────────────────┤
    │                                                          │
    └─→ dwt_sf_estimation(iq, bw_candidate, t_offset) ───────┤
          for sf_hat in 7..12:                                │
            ref = down_chirp(sf_hat, bw_candidate)           │
            dechirped = iq[t_offset:t_offset+n_use] × ref    │
            spectrum = rfft(dechirped, n=n_use)              │
            score[sf_hat] = peak / mean                      │
          → sf_candidate ────────────────────────────────────┤
                                                              │
    Confidence gate (threshold_low=0.4, threshold_high=0.7) ←┘
    holdoff = 100ms
    → PDU {sf, bw, confidence, method='psd+dwt', sst_used=False, ...}
```

## Known Issue (non-blocking)

**SF7/BW=812k @ SNR=−14 dB = 0/50 trials.**

- `n_sym(SF7, BW=812k) = 4,840` — найменший буфер серед усіх 24 комбінацій.
- При SNR=−14 dB BW margin між 812k та 1625k мала → bw_candidate=1625k (wrong).
- Wrong BW → wrong ref chirp → random score → sf≠7 → pair fails.
- Загальний pair=95.8% >> 78% target → PASS не відхиляється.
- Зафіксовано для майбутнього виправлення (підвищена кількість символів або dual-BW hypothesis).
