## Задача на Крок 1 — 2D OS-CFAR специфікація

### Що потрібно визначити:

```
Алгоритм:    2D OS-CFAR (Ordered Statistic CFAR)
Параметри:
  - k        ← порядкова статистика (яке значення з вікна)
  - N_ref    ← розмір reference window
  - N_guard  ← розмір guard cells
  - Pfa      ← бажана ймовірність хибної тривоги (ціль ≤ 1%)
  - Pd       ← ймовірність виявлення (ціль?)
Середовище:  AWGN + chirp-сигнал (синтетика)
Throughput:  ціль ≥ 80 MS/s
```
