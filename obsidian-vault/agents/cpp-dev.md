## Вимоги до коду

```cpp
// cfar2d.hpp — мінімальний інтерфейс
class CFAR2D {
public:
    CFAR2D(int N_ref, int N_guard, int k);
    std::vector<Detection> process(const Matrix2D& input);
    double throughput_ms() const; // MS/s
};
```

- Стандарт: C++17 або вище
- Без зовнішніх залежностей (крім STL та опційно Eigen)
- Компілюється: `g++ -O3 -mavx2`
