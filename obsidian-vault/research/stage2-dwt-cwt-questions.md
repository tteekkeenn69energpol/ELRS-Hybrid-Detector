# 45 Глибоких Питань для Stage 2 (DWT + CWT) — Доппитання

Ці 45 питань призначені для аналізу архітектури Stage 2 через NotebookLM та слугують чек-листом валідності проектних рішень.

---

## Розділ 1: Архітектура та Принципи

1. **MRA та SF:** Як вейвлет-коефіцієнти різних рівнів корелюють з SF у LoRa? Чи існує лінійна залежність між рівнем декомпозиції та логарифмом SF?
2. **CWT vs STFT:** Чому CWT з Morlet дає кращу роздільність для chirp rate, ніж STFT?
3. **SST:** Чи виправдовує навантаження Synchrosqueezed CWT при Low SNR?
4. **Каскад:** DWT для грубого SF → CWT для BW, чи виконувати паралельно?
5. **Інваріантність до шуму:** Поведінка вейвлетів при SNR < -15 dB порівняно з енергетичними методами. Чи є "поріг провалу"?
6. **Transients:** Чи можуть вейвлети краще за FFT детектувати короткі преамбули?
7. **CFO вплив:** Як Carrier Frequency Offset впливає на CWT точність? Пік зсувається чи розмивається?
8. **Складність:** O(N) для CWT порівняно з FFT (N log N)? Вмістимось у 15 мс?
9. **Масштабування:** Чи працює один детектор для вузькосмугових (SF12, BW 125kHz) та широкосмугових (SF7, BW 500kHz)?

---

## Розділ 2: Алгоритми та Математика

10. **Вибір вейвлету:** Symlet/Daubechies для DWT чи Morlet для CWT — чому?
11. **SF через енергію:** Чи надійне співвідношення енергії в Detail коефіцієнтах для розрізнення SF7 vs SF8?
12. **Scale ↔ BW:** Точна математична формула конвертації Scale → BW для LoRa CSS?
13. **Фаза:** Чи потрібно враховувати фазу вейвлет-коефіцієнтів для точнішої оцінки ToA?
14. **LFSR та передбачення:** Чи можна через DWT-виявлену периодичність передбачити наступний Frequency Hop?
15. **Циклостаціонарність:** Чи можна витягти ознаки SCF прямо з вейвлет-спектра?
16. **ToA оцінка:** Як визначити початок чирпа з похибкою < 1 мкс через гребінь скейлограми?
17. **Порівняння з MF:** Наскільки вейвлет-оцінка програє ідеальному Matched Filter за SNR?
18. **Адаптивні пороги:** Як автоматично встановлювати поріг для CWT при динамічному рівні шуму?

---

## Розділ 3: Реалізація та Оптимізація (CUDA)

19. **GPU Acceleration:** Ефективна реалізація CWT на CUDA через convolutions?
20. **Shared Memory для DWT:** Уникнення bank conflicts при рівневому розкладанні?
21. **Дискретизація Scale:** Чи достатньо 10–20 дискретних масштабів замість повного спектра?
22. **Fixed-Point:** Ядро вейвлету в INT16 без критичної втрати точності для Edge AI?
23. **Паралелізація:** Розпаралелювання різних Scale по різних CUDA блоках?
24. **FPGA ресурси:** Скільки DSP Slices для CWT блоку на Artix-7?
25. **Бібліотеки:** PyWavelets + Numba JIT чи кастомний CUDA kernel для 100+ MS/s?
26. **Boundary effects:** Як обробляти крайові ефекти при безперервному потоці?
27. **Pipeline Latency:** End-to-end від семплу до оцінки BW/SF?

---

## Розділ 4: Тестування та Валідація

28. **Синтетика vs Реальність:** Різниця вейвлет-картин синтетичних та реальних LoRa сигналів?
29. **Метрики:** Цільова RMSE для chirp rate при -12 dB SNR?
30. **CW jamming:** Поведінка детектора при вузькосмугових завадах?
31. **Collision:** Розрізнення двох одночасних сигналів з різними SF?
32. **False Positives:** Pfa на шумові сплески з chirp-подібною структурою?
33. **Ground Truth:** Як валідувати SF/BW якщо передавач з невідомими параметрами?
34. **Doppler:** Точність оцінки при різкій зміні несучої частоти?
35. **Візуалізація:** Real-time скейлограма для оператора?

## Розділ 5: Інтеграція та Масштабування

36. **Інтерфейс Stage 3:** Формат передачі SF, BW, ToA? (PMT, JSON, Shared Memory)
37. **Fallback:** Що робити якщо confidence < 0.7?
38. **Адаптивність:** Швидкість перекалібрування при зміні конфігурації завад?
39. **Ресурси:** Відсоток GPU часу Stage 2 відносно загального pipeline?
40. **Multi-antenna:** Різниця фаз вейвлет-коефіцієнтів для AOA?
41. **Збереження стану:** Kalman filter для згладжування результатів?
42. **Hot-reload:** Зміна материнського вейвлету без перезапуску?
43. **Масштабування смуги:** Перехід від 915 MHz до 2.4 GHz?
44. **"Blindness":** Чи потрібне хоч мінімальне апріорне знання (це LoRa-подібний сигнал)?
45. **Енергоефективність:** Вимикати Stage 2 при низькій енергії сигналу (Stage 1)?
